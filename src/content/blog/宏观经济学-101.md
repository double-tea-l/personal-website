###### 1. GDP

定义：一段时间内某国生产的所有物品及劳务的价值，假设全部支出 = 全部收入。

GDP = C + I + G + NX = 消费 + 投资 + 政府购买 + 净出口。即个人/企业/政府的支出 + 国家的收入。

* 名义 GDP: 按照当年市场价格计算的 GDP。
* 实际 GDP: 按照选定的基年价格计算的 GDP; 因为名义 GDP 的部分上涨有可能是价格（通货膨胀）带来的，按照基年价格计算可以剥离通胀因素。
* GDP 平减指数 = 名义 GDP / 实际 GDP, 可以理解为 inflated price. 
* 通胀率 (可以理解为 inflated price y/y) = GDP 平减指数 y/y

GDP 可以衡量一国的经济实力（生产总值），但是无法直接衡量国民的生活质量，社会福利，和贫富差距等其它影响幸福水平的因素。

###### 2. CPI

定义: GDP 是衡量一国生产总值，cpi 是衡量家庭消费支出的价格。工业生产的价格变动不会记在 cpi。cpi 是政府选定的“一揽子”消费品的价格，用来描述个人和家庭消费支出的价格，包括衣食住行等主要方面。

（find the breakdown）

CPI = 当年价格 / 基年价格

* 局限性：只描述价格，不描述质量。如果同样的价格下，质量增加/较少，都不会被捕捉。一篮子有滞后性，出现替代品和新产品，不会被及时反应，居民体感价格一般比cpi要低。
* 通货膨胀率 = cpi y/y. Cpi计算的通胀更能反应普通居民生活感受到的物价变化。gdp 计算的通胀率反映包括生产制造在内的价格变化。
* 名义利率 = nominal interest rate
* 实际利率 = nominal interest rate - inflation rate 

通胀时，名义利率 > 实际利率，紧缩时相反。

###### 3. 资本与生产力

* 生产率 （productivty)：每单位劳动投入所生产的物品和劳务的数量。一国的生活水平，取决于它生产物品与劳务的能力。
* 物质资本 + 人力资本 + 技术知识 + 自然资源
* 资本<>生产率的 diminishing return （没太懂）
* 追赶效应：起点低进步快，因为出于 curve 更 steep 的位置。

###### 4. 金融市场与利率

* 金融体系：将一个人的储蓄与另一个人的投资匹配的机构
* 金融市场：有闲钱的人向需借款的人提供资金的机构。
* 债券：借款凭证；期限/信用风险/税收待遇
* 股票：企业部分所有权；道琼斯/标普500/纳斯达克
* 金融中介机构：有闲钱的人间接向借款的人的提供资金的机构。
* 银行
* 共同基金：多元化投资/专业化管理
* 国民储蓄 = S = I = 私人储蓄 + 公共储蓄
* 私人储蓄 = y - t - c = yield (GDP) - tax - consumption
* 公共储蓄 = t - g = tax - government purchase
* 预算盈余 = tax - government purchase > 0 

政府政策对利率和储蓄/投资的影响：

1/ 鼓励储蓄 -> 可贷资金增加 -> 利率下降 -> 投资增加

2/ 鼓励投资 -> 可贷资金减少 -> 利率上升 -> 储蓄增加 & 投资减少

3/ 政府赤字下降 -> 可贷资金减少 -> 利率上升 -> 投资减少

挤出效应：由政府借款引起的投资减少
